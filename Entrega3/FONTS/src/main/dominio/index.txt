-----------------Directorio dominio-----------------

Este directorio contiene todos los archivos .java de las clases
del dominio. Para compilar el programa usar el Makefile de la
carpeta src en FONTS/. Para mas informacion sobre las clases del
dominio mirar el documento "2aEntregaDocumentacion".

Elementos del directorio
- algoritmos/:
	Contiene las clases de los algoritmos

- ed/:
	Contiene una unica clase nodo usada para los algoritmos

- Alfabeto.java:
	Clase del dominio perteneciente a Alfabeto

- ComprobarExcepciones.java:
	Clase del dominio perteneciente a ComprobarExcepciones

- CtrlAlgoritmo.java:
	Clase del dominio perteneciente a Controlador de algoritmo

- CtrlEntrada.java:
	Clase del dominio perteneciente a Controlador de entrada

- Input.java:
	Clase del dominio perteneciente a Input

- ListaPalabras.java:
	Clase del dominio perteneciente a Lista de palabras

- Playout.java:
	Clase del dominio perteneciente a Playout

- Teclado.java:
	Clase del dominio perteneciente a Teclado

- Texto:
	Clase del dominio perteneciente a Texto

- TLP:
	Clase del dominio perteneciente a TLP

- UtilesAlgoritmo:
	Clase del dominio perteneciente a Utiles Algoritmo
